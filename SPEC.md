# Tesla Control

## Spec Sheet:

* Goal - Allow AI agents to gain access to Tesla vehicles. A smart assitant (voice or text based) for the vehicle. Not meant for agent based driving and time-critical operations.

* Target Users - Tesla Vehicle Owners

* Core Capabilities:
    1. Gathering vehicle system information.
    2. Basic control of lights, AC, doors.
    3. Setting a driving destination.
    4. Accessing the main vehicle panel.
